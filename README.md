# SKR-YungBro-EPK | M3-Hackathon-2021

Official 24/7 Continuous Music Hub and Electronic Press Kit for **YUNGBRO**.

## 🧬 Concept
A persistent digital broadcast engine. This site is designed to never stop. It serves as both a high-retention music player and a professional gateway for industry professionals to access YungBro's media assets.

## 🛠 Features
- **Zero-Latency Loop:** Configured to cycle YungBro's lead single indefinitely.
- **EPK Integration:** Direct access to press photos, bios, and booking info.
- **Streaming Grid:** One-tap links to all major DSPs (Spotify, Apple, etc.).
